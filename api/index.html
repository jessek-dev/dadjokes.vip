Dad Jokes API

Free JSON feed with 5,000+ dad jokes. No API key required.

Get Dad Jokes as JSON

dadjokes.vip provides a free, publicly accessible JSON feed of dad jokes. Use it to build bots, add jokes to your website, or just make your coworkers groan on Slack. No authentication, no rate-limit keys, no sign-up.

The feed follows the JSON Feed 1.1 specification, so it works out of the box with any feed reader or HTTP client.

Endpoints

Curated Jokes Feed

GET

https://dadjokes.vip/jokes.json

Returns the curated joke collection in JSON Feed format. Each joke includes a unique ID, the joke text, categories, and timestamps.

Full Dataset

GET

https://dadjokes.vip/jokes-full.json

The complete dataset including all jokes. Larger file size — best for bulk downloads or one-time imports rather than repeated fetching.

Response Format

Each joke item in the feed looks like this:

{
  "id": "https://dadjokes.vip/joke/MJ000001",
  "url": "https://dadjokes.vip/joke/MJ000001",
  "title": "Wife: Honey I'm pregnant...",
  "content_text": "Wife: Honey I'm pregnant. Me: Well... what do we do now? Wife: Well, I guess we should go to a baby doctor. Me: Hm... I think I'd be a lot more comfortable going to an adult doctor.",
  "date_published": "2024-01-01T00:00:00Z",
  "tags": ["Wife", "dad jokes", "humor"],
  "author": {
    "name": "Dad Jokes",
    "url": "https://dadjokes.vip"
  }
}

Key Fields

  • id / url — Unique permalink for the joke
  • title — The joke setup (or short version)
  • content_text — The full joke including the punchline
  • tags — Categories the joke belongs to
  • date_published — When the joke was added

Quick Start

Fetch a random joke with curl and jq:

# Fetch the feed and pick a random joke
curl -s https://dadjokes.vip/jokes.json \
  | jq '.items | .[length * random | floor] | .content_text'

Or in JavaScript:

const res = await fetch('https://dadjokes.vip/jokes.json');
const feed = await res.json();
const joke = feed.items[Math.floor(Math.random() * feed.items.length)];
console.log(joke.content_text);

In Python:

import requests, random

feed = requests.get("https://dadjokes.vip/jokes.json").json()
joke = random.choice(feed["items"])
print(joke["content_text"])

Rate Limits & Attribution

The JSON feed is a static file served from a CDN. There are no formal rate limits, but please be a good citizen:

  • Cache responses locally. The feed does not change every minute. Fetching once per day (or even once per hour) is more than enough.
  • Do not hammer the endpoint. Since it is a static file, aggressive polling just wastes bandwidth for everyone.

Attribution

The data is free to use. All we ask is that you include a link back to dadjokes.vip or the App Store listing somewhere visible. A small "Jokes from dadjokes.vip" credit is perfect.
